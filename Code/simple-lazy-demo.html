How It Works

This follows the exact pattern you requested:

window.addEventListener('scroll', function () {
    if (window.scrollY > 500) {
        // Load sections with data-lazy-load attributes
        // "mobile" - loads on mobile only
        // "desktop" - loads on desktop only  
        // "both" - loads on both devices
    }
});

Usage: Add data-lazy-load="mobile" or data-lazy-load="desktop" to any section.
